# springboot-jpa-web-application-optimized

## Repository Status
This repository represents the optimized version of the baseline web application.  
Key improvements include performance optimization and API-centric refactoring.

## Related Repository
- Baseline Implementation: https://github.com/hyejin0916/springboot-jpa-web-application

## Features & Improvements
- 성능 최적화: 쿼리 튜닝, 캐싱 적용, 효율적인 DB 접근
- API 중심 설계: 뷰 렌더링을 대체하는 RESTful 엔드포인트 구성
- JPA 엔티티 설계 개선 및 DTO 활용
- 검증, 예외 처리, 트랜잭션 관리 강화
- 서비스 레이어 모듈화 및 아키텍처 구조 개선

## Notes
- This repository builds upon the baseline web application in the springboot-jpa-web-application repository.
- The baseline repository remains unchanged for reference.
